Support for the nRF9160DK board

General information

The nRF9160DK is a devboard based on nRF9160 MCU which offers LTE-M and NB-IoT connectivity.

The board features four LEDs, four user buttons/switches and a reset button.

Flash the board

The board is flashed using JLink or nrfjprog software. Programs needs to be installed.

The process is automated in the usual make flash target.

Accessing STDIO via UART

The STDIO is directly accessible via the USB port. On a Linux host, it's generally mapped to /dev/ttyACM0.

Use the term target to connect to the board serial port

make BOARD=nrf9160dk -C examples/hello-world term
