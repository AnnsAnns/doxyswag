Detailed Description

Support for the Microchip SAM R34 Xplained Pro and WLR089 Xplained Pro

Overview

The SAMR34 Xplained Pro is a compact evaluation board by Microchip featuring a SAMR34J18 SoC. The SoC includes a SAML21 ARM Cortex-M0+ micro-controller bundled with Semtech's SX1276, a 868/915 MHz LoRa compatible radio. For programming the MCU comes with 40Kb of RAM and 256Kb of flash memory.

Hardware

An evaluation board for the radio certified WLR089 module exists as the WLR089 Xplained Pro, it is compatible with the samr34-xpro.
