The Sipeed Longan Nano TFT board is a version of the Sipeed Longan Nano development board that is equipped with a TFT display with the following on-board components:

  • GD32VF103CBT6 RISC-V MCU @108MHz
  • USB Type C
  • TF card slot
  • 3 user LEDs
  • 0.96" TFT display 160 x 80 pixel
Sipeed Longan Nano

Detailed information about the board configuration and flashing can be found in the Sipeed Longan Nano documentation.
