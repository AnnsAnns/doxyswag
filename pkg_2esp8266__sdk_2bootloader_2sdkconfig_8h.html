Detailed Description

RIOT-OS modification of the bootloader SDK configuration.

The bootloader build of the ESP8266 SDK needs some settings from the SDK configuration. These are normally generated by the menuconfig in the vendor SDK.

Some of these parameters are configurable by the application. For example, the UART baudrate used by the console and the verbose level of the bootloader.
