Detailed Description

Mapping from MCU pins to Arduino pins.

You can use the defines in this file for simplified interaction with the Arduino specific pin numbers.

#define ARDUINO_SPI_D11D12D13   SPI_DEV(0)

SPI_DEV(0) is connected to D11/D12/D13 for most Nucleo-64 boards.

This can be overwritten in boards/nucleo-<foobar>/include/periph_conf.h by providing a custom ARDUINO_SPI_D11D12D13.
