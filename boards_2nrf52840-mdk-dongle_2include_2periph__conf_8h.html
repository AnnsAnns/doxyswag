UART configuration

This board does not have explicit UART pins.

These are set as UART 0 to provide an easy serial debug port when not using (or debugging) USB.

PWM configuration

For the nRF52840-mdk-dongle board, the PWM0 module is set to drive the RGB LED.

Other PWM outputs are not configured.
